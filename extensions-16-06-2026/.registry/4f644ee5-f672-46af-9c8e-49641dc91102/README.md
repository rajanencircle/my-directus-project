# Introduction
This extension provides a button interface that performs a "Save and Stay" command when clicked.

**Details**
- The button is disabled if there are no edits to the item.
- A spinner is displayed while saving.

First used in Quickify AI. Learn More: https://quickify.ai

# Installation
```
npm i @quickifyai/directus-extension-save-and-stay-button
```
OR

Search for "Save and Stay" in the Directus Marketplace 

# Get Started
1. Go to **Settings** -> **Data Model**, in a **Collection** choose to **Create Field in Advanced Mode**.
2. Select **Presentation and Aliases** 
3. In the **Interface** section, choose the **Save and Stay Button**.
4. Add a **Label**, **Icon**, and **Type** as usual.
